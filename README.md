# NodusAI MCP Server

> AI-Powered Signals for Prediction Markets — accessible to any AI agent via MCP.

AI agents connect to this server to get Oracle signals for [Polymarket](https://polymarket.com) and [Kalshi](https://kalshi.com) prediction markets. Signals are generated by Gemini 2.5 Flash with real-time web grounding. Payment is handled natively via the x402 protocol (USDC on Base).

---

## Payment model

**One simple price: $1 USDC = 3 Oracle signal queries.**

- **Networks:** Base, Ethereum, Avalanche (any EVM chain)
- **Token:** USDC
- **Protocol:** x402 / EIP-3009 `TransferWithAuthorization`
- **Non-custodial:** NodusAI never holds funds. Payments go directly on-chain.

Pay $1 → get a session token → use it for 3 queries. Need more? Pay another $1.

---

## How it works

```
AI Agent  ──MCP──►  NodusAI MCP Server  ──►  nodusai.app/api/prediction
                          │                        │
                          │                   Gemini 2.5 Flash
                          │                   + Web grounding
                          │
                          └──►  Query Registry (every signal logged)
```

1. Agent calls `nodus_get_payment_requirements` to see what to pay
2. Agent signs $1 USDC `TransferWithAuthorization` (EIP-3009) with their Base wallet
3. Agent calls `nodus_get_signal` with the signed payment header
4. Server validates payment → Oracle queries Gemini 2.5 Flash with live web grounding
5. Signal returned: probability, confidence, reasoning, grounding sources
6. Session token issued — use `nodus_query_with_session` for 2 more free queries
7. All 3 queries used? Pay another $1 for 3 more.

## Available tools

| Tool | Description | Cost |
|------|-------------|------|
| `nodus_pricing` | View pricing: $1 USDC = 3 queries | Free |
| `nodus_get_payment_requirements` | Get EIP-3009 payment details for a market | Free |
| `nodus_get_signal` | **Pay $1 USDC → get Oracle signal + session token (3 queries)** | $1 USDC |
| `nodus_query_with_session` | Use remaining queries from your session | Free |
| `nodus_verify_signal` | Audit grounding sources of a past signal | Free |
| `nodus_query_history` | Your recent queries from the registry | Free |
| `nodus_admin_stats` | Platform-wide stats | Admin |
| `nodus_mock_payment` | Generate mock $1 USDC payment header for dev | Dev only |

## Deploy in 5 minutes (no local setup needed)

### Option 1 — Railway (recommended)

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/new/template)

1. Fork this repo on GitHub
2. Go to [railway.app](https://railway.app) → **New Project** → **Deploy from GitHub repo**
3. Select your fork
4. Add environment variable: `NODUSAI_API_BASE` = `https://nodusai.app`
5. Railway auto-detects `railway.json` and deploys
6. Copy your Railway URL (e.g. `https://nodusai-mcp.up.railway.app`)

That's it. Your MCP server is live.

### Option 2 — Render (free tier)

1. Fork this repo
2. Go to [render.com](https://render.com) → **New Web Service** → connect your fork
3. Set **Build command:** `npm install` and **Start command:** `node src/server-http.js`
4. Add env var: `NODUSAI_API_BASE=https://nodusai.app`
5. Deploy

### Custom JS agent

```bash
npm install @modelcontextprotocol/sdk
```

```javascript
import { Client } from "@modelcontextprotocol/sdk/client/index.js";
import { SSEClientTransport } from "@modelcontextprotocol/sdk/client/sse.js";

const client = new Client({ name: "my-agent", version: "1.0.0" }, { capabilities: {} });
await client.connect(new SSEClientTransport(new URL("https://YOUR-DEPLOYMENT-URL/sse")));

// Step 1 — check pricing
const pricing = await client.callTool({ name: "nodus_pricing", arguments: {} });

// Step 2 — pay $1 USDC, get signal + session token (3 queries total)
const result = await client.callTool({
  name: "nodus_get_signal",
  arguments: {
    marketUrl:      "https://polymarket.com/event/...",
    xPaymentHeader: "<base64-eip3009-signed-$1-usdc-auth>",
    walletAddress:  "0xYourBaseWallet",
  }
});
// result.session.token  → use for next 2 free queries
// result.session.queriesRemaining  → 2

// Step 3 — use session for free query 2 of 3
const result2 = await client.callTool({
  name: "nodus_query_with_session",
  arguments: {
    marketUrl:    "https://polymarket.com/event/...",
    sessionToken: result.session.token,
  }
});

// Step 4 — use session for free query 3 of 3
// After this, pay another $1 for 3 more queries
```

### Custom Python agent

```bash

```

```python
from mcp.client.sse import sse_client
from mcp import ClientSession

async with sse_client("https://YOUR-DEPLOYMENT-URL/sse") as (read, write):
    async with ClientSession(read, write) as session:
        await session.initialize()

        # Check pricing — $1 USDC = 3 queries
        pricing = await session.call_tool("nodus_pricing", {})

        # Pay $1 USDC, get signal + session token
        result = await session.call_tool("nodus_get_signal", {
            "marketUrl":      "https://kalshi.com/markets/...",
            "xPaymentHeader": "<base64-eip3009-signed-$1-usdc-auth>",
            "walletAddress":  "0xYourBaseWallet",
        })
```
